# CAIN Africa Homepage (Assessment Project)

This is a simple responsive homepage I built as part of a technical assessment.

The goal was to create a clean and user-friendly layout that reflects the mission of CAIN Africa, while keeping the design simple and functional.

---

## 🔧 What I Built

The homepage includes:

- A header with navigation links
- A hero section with a clear message
- An about section explaining the organization
- A programs section showing key focus areas
- A footer with a back-to-top link

---

## 🎯 Approach

I focused on keeping the design:

- Clean and easy to read
- Simple but modern
- Structured in a way that is easy to maintain

I also made sure the content aligns with CAIN Africa’s focus on digital skills, youth empowerment, and community impact.

---

## 📱 Responsiveness

The layout adjusts for smaller screens using media queries.  
Navigation and sections stack properly on mobile devices.

---

## 🛠️ Tools Used

- HTML5
- CSS3
- Flexbox
- Media Queries
